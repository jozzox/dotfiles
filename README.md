# 🛠️ Meine Dotfiles

Dieses Repository enthält meine persönlichen Dotfiles für eine konsistente und minimalistische Arbeitsumgebung.
Der Fokus liegt auf einem harmonisierten Look & Feel.

---

## 🧰 Systemumgebung

- 🐧 **OS**: Fedora 42
- 🖥️ **Desktop/WM**: KDE4 Plasma
- 🧠 **Shell**: zsh
- 🖥️ **Terminal**: Kitty
- 🎨 **Theme**: TokyoNight
- 🔤 **Fonts**: Cascadia
- 🖌️ **Icons**: Papirus-Dark

---

## 📁 Dotfile-Struktur (`~/.config/` basiert)

```text
dotfiles/
        ├── kitty/             → Terminal-Konfiguration
        ├── zsh/               → Shell (zshrc, Aliase, Plugins)
```

---

## 📦 Benötigte Pakete (abhängig von deinem Setup)

| Bereich        | Pakete                                                  |
|----------------|---------------------------------------------------------|
| Terminal       | `kitty`                                                 |
| Shell          | `zsh`, `zinit plugin Manager`                           |
| Fonts/Icons    | `nerd-fonts`, `papirus-icon-theme`                      |
| FileManger     | `Yazi`                                                  |
| FileManger     | `Neovim`,`NVIM v0.11.2`                                 |

---

## 🎨 Einheitliches Theme-Design

| Komponente | Theme / Stil           | Beschreibung                            |
|------------|------------------------|-----------------------------------------|
| GTK        | [TokyoNight]           | Konfiguration in `~/.config/gtk-3.0/`   |
| Terminal   | [Tokyonight]           | von Folke/folke.nvim                    |
| Icons      | [Nerd-Fonts]           | Systemweite Icons                       |
| Fonts      | [CascadiaMono]         | Für Terminal, Editor und UI-Elemente    |
